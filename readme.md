# Jestlang

Jestlang é uma linguagem de programação minimalista focada em backend, inspirada em Python, Lua e Go.

Criada para ser simples, rápida, produtiva e divertida de programar.

## Visão

A Jestlang nasceu com o objetivo de criar uma experiência de programação:

- simples
- moderna
- limpa
- rápida de escrever
- fácil de aprender
- focada em backend
- open source

A ideia é remover complexidade desnecessária e deixar o código mais natural.

## Filosofia

A Jestlang prioriza:

- menos palavras-chave
- menos boilerplate
- sintaxe limpa
- produtividade
- legibilidade
- simplicidade

O objetivo é fazer programar parecer leve.

## Features Atuais

- Variáveis
- Funções
- Loops
- If / Else
- Tipagem dinâmica
- Execução via CLI
- Lexer próprio
- Parser próprio
- AST própria
- Interpretador próprio
- Arquivos ".jest"

## Exemplo

```jest
print("Jestlang Boot")

fn greet(name):
    print("Hello")
    print(name)

users = 5

for i in users
    greet("Lucas")
```
